# aianalyze
This repository contains a Colab notebook `boan_data_pipeline.ipynb` that downloads and indexes MLTMA press releases into SQLite and Faiss.

## Running in Colab
1. Open <https://colab.research.google.com>.
2. Go to **File > Open notebook > GitHub**.
3. Paste this repository's URL and select `boan_data_pipeline.ipynb`.
4. Edit the configuration cell to set your API key and folders.
5. Run the notebook to download PDFs and build the index.
